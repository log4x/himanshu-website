# Design Philosophy - Himanshu Chauhan Portfolio

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a1d29) - Professional, trustworthy, technical depth
- **Secondary**: Warm Copper (#c47f3e) - Innovation, data science warmth, achievement highlights
- **Accent**: Soft Teal (#4a9b8e) - Growth, learning, continuous improvement
- **Neutral**: Light Gray (#f8f9fa) - Clean backgrounds, readability
- **Text**: Charcoal (#2c3e50) - High contrast, professional readability

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings, adds sophistication
- **Body Font**: "Inter" - Clean, modern sans-serif for excellent readability
- **Code Font**: "JetBrains Mono" - Technical content, skill listings
- **Hierarchy**: Large bold headings, medium subheadings, comfortable body text

### Visual Language
- **Minimalist Aesthetic**: Clean lines, generous white space, focused content
- **Technical Precision**: Grid-based layouts, structured information architecture
- **Professional Warmth**: Balanced use of color to avoid cold, sterile feeling
- **Data-Driven**: Visual elements that suggest analytics, networks, systems thinking

## Visual Effects & Styling

### Background Effects
- **Primary**: Subtle animated network nodes connecting in background using p5.js
- **Secondary**: Gentle gradient overlay with floating geometric shapes
- **Consistency**: Same background treatment across all pages for cohesive experience

### Interactive Elements
- **Hover States**: 3D tilt effects on cards, subtle shadow expansion
- **Transitions**: Smooth 300ms easing for all interactive elements
- **Micro-animations**: Skill bars filling on scroll, achievement badges appearing

### Component Styling
- **Cards**: Clean white backgrounds with subtle shadows, rounded corners
- **Buttons**: Gradient backgrounds with hover state transformations
- **Timeline**: Vertical line with connected nodes, smooth reveal animations
- **Charts**: Consistent color scheme with interactive hover details

### Header Effects
- **Navigation**: Fixed header with blur backdrop, smooth color transitions
- **Hero Section**: Animated background with typewriter text effect for name
- **Scroll Indicator**: Subtle progress bar showing page scroll position

### Responsive Design
- **Mobile-First**: Touch-friendly interactions, optimized for all screen sizes
- **Breakpoints**: 768px (tablet), 1024px (desktop), 1440px (large screens)
- **Typography Scale**: Responsive font sizing for optimal readability

### Animation Library Usage
- **Anime.js**: Page transitions, skill visualizations, timeline animations
- **ECharts.js**: Interactive radar charts for skills, certification progress
- **Splide.js**: Project showcase carousels, testimonial sliders
- **p5.js**: Background network animation, data visualization effects

### Accessibility Features
- **Color Contrast**: Minimum 4.5:1 ratio for all text elements
- **Focus States**: Clear keyboard navigation indicators
- **Screen Reader**: Proper ARIA labels and semantic HTML structure
- **Motion Preferences**: Respect user's reduced motion preferences
